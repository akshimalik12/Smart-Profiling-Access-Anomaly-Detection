# Contributing to Smart Profiling

Thanks for your interest! Please follow these guidelines:

## Get started
1. Fork the repo
2. Create a branch: `git checkout -b feat/my-feature`
3. Implement changes, add tests where relevant
4. Commit with clear message and push

## Coding style
- JavaScript/React: follow Prettier formatting (recommended)
- Python: follow PEP8 + use black/flake8 if possible

## Pull requests
- Create PR against `main`
- Describe what problem is solved and include steps to reproduce
- Link any related issues

## Tests & CI
- Try to keep CI green; update workflows in `.github/workflows/` where needed.
